# catch-mind

RealTime Drawing Game with Gulp, SocketIO and Node

# setup

1. npm intall --global yarn
2. yarn init
3. yarn add nodemon(devDependency)
4. yarn add express
5. yarn add socket.io
6. yarn add @babel/node
7. yarn add @babel/core
8. yarn add @babel/preset-env
9. server.js
10. .babelrc
11. package.json
12. yarn dev:server
